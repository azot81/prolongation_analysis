# Демонстрационный проект Data Analyst

Данный проект является презентацией навыков и умений дата-аналитика по программированию с помощью языка **Python** и использованию библиотек **Numpy**, **Pandas**, **Matplotlib**, **Seaborn**, **Plotly**.
---

В качестве данных использованы обезличенное техническое задание заказчика, прописанное в файле **"ТЗ Data Analyst.docx"**, а также два исследуемых дата-сетов **"financial_data.csv"** и **"prolongations.csv"**.

## Проект состоит из четырёх частей:
### <center> 1. Загрузка необходимых библиотек и дата-сетов </center>
### <center> 2. Создание таблиц, необходимых для аналитики </center>
### <center> 3. Создание и экспорт файлов аналитического отчёта </center>
### <center> 4. Визуализация данных для аналитического отчёта </center>

#### При преобразовании данных в проекте показано умение обращаться с:
* функциями Python
* циклами Python
* преобразованием типов данных
* добавлением и удалением столбцов библиотеки Pandas
* соединением таблиц библиотеки Pandas

#### В проекте демонстрируется умение пользоваться визуализацией для анализа данных с построением графиков:
* столбчатая диаграмма
* многоуровневая столбчатая диаграмма
* круговая диаграмма
* тепловая карта
* линейный график

#### Использованные инструменты и библиотеки:
* Python (3.9.10)
* Numpy (2.0.2)
* Pandas (2.3.2)
* Matplotlib (3.9.4)
* Seaborn (0.13.2)
* Plotly (6.3.1)
